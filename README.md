# Drvstella — Minimalist Photography Portfolio

A dark-themed, responsive photography portfolio built with React + Vite. Features a bento grid layout, interactive lightbox, category filters, and Exif camera details.

## Live Site

🌐 [drvstella.pages.dev](https://drvstella.pages.dev)

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Database:** Supabase
- **Hosting:** Cloudflare Pages

## Run Locally

```bash
npm install
npm run dev
```
